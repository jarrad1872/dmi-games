## Core Mechanics (from research)

### Controls
- **Swipe Left/Right**: Switch between 3 lanes
- **Swipe Up**: Jump over obstacles
- **Swipe Down**: Slide under barriers
- **Instant response**: No lag, immediate feedback

### Power-Ups (5 main types)
1. **Jetpack** - Fly above tracks, collect aerial coins
2. **Super Sneakers** - Higher/longer jumps
3. **Coin Magnet** - Attracts coins from all lanes
4. **2x Multiplier** - Doubles score for duration
5. **Hoverboard** - Crash protection (one-time save)

### Obstacles
- Trains (stationary and moving)
- Barriers and barricades
- Tunnels (must slide)
- Gaps (must jump)

### UI Elements
- Score counter (top left)
- Coin counter (top right)
- Distance meter
- Power-up indicators
- Pause button

## DMI HEAT RUNNER Adaptation

### Theme: Construction Site Runner
- Player: Construction worker with hard hat
- Setting: Active construction zone
- Visual style: Industrial, orange/yellow safety colors

### DMI Power-Ups
1. **Hard Hat** = Crash protection (like Hoverboard)
2. **Safety Vest** = 2x Multiplier (visibility bonus)
3. **Drill Bit Boost** = Speed boost + magnet
4. **Tool Belt** = Coin magnet
5. **Jetpack** = Keep as-is (construction version)

### DMI Obstacles
- Concrete barriers
- Scaffolding
- Equipment (forklifts, mixers)
- Rebar stacks
- Wet cement zones (slow down)

### Tool Drop Integration
- Appears after milestone scores
- Features real DMI products
- "Stay safe on the job site" messaging
