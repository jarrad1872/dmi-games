## DMI Integration Points

### Natural Product Placement
- DMI core bits as upgrades
- Diamond segments for deeper drilling
- Real product catalog as "shop"

### Tool Drop Triggers
- On first upgrade
- Every depth milestone (100m, 500m, 1000m)
- On rare core discovery

### Upgrade System
- Standard bit → DMI Segmented → DMI Turbo → Custom Core Bit
