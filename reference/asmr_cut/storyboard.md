# ASMR CUT - Storyboard

## Reference Game
**Name**: ASMR Slicing / Soap Cutting Simulator
**Platform**: Mobile
**Why This Reference**: Extremely satisfying tactile feedback, simple mechanics, addictive progression

---

## Core Loop

### 1. Entry Point
**Frame**: frame_001.png
**Description**: Object selection screen - choose what to slice
**Key Elements**:
- Grid of sliceable objects
- Unlock indicators
- Daily reward prompt

### 2. Main Gameplay
**Frame**: frame_002.png - frame_005.png
**Description**: Touch and drag to slice through objects
**Key Elements**:
- Object on cutting board
- Slice trail/indicator
- Satisfying particle effects on cut
- Object separation physics

### 3. Reward/Feedback
**Frame**: frame_006.png
**Description**: Slice completion celebration
**Key Elements**:
- Coin/score popup
- Star rating
- Satisfying sound cue

### 4. Progression
**Frame**: frame_007.png
**Description**: Unlock new objects and blades
**Key Elements**:
- New item unlocked notification
- Collection progress
- Blade upgrade shop

---

## DMI Integration Points

### Natural Product Placement
- Diamond blades as premium cutting tools
- Core bits unlock special "drill through" mode
- DMI branding on blade selection screen

### Tool Drop Triggers
- After completing level 5 (first hook)
- Every 10 levels thereafter
- On blade upgrade screen

### Upgrade System
- Standard blade → DMI Segmented blade → DMI Turbo blade
- Each upgrade cuts faster/cleaner

---

## Feel Targets

### Visual
- Clean, bright colors
- Satisfying particle explosions on cut
- Smooth slice trail following finger
- Object pieces fall with weight

### Audio
- Crisp slicing sounds
- Coin collection jingles
- Upgrade whoosh sounds

### Haptic
- Light vibration on slice start
- Strong vibration on completion
