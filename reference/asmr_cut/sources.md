## Design Inspiration

### Core Loop (from references)
1. Present sliceable object
2. Player draws cut lines
3. Pieces fall with physics
4. Collect pieces into target
5. Star rating based on efficiency
6. Earn coins for upgrades

### Visual Style Notes
- Soft, pastel color palette
- 3D rendered objects with simple textures
- Satisfying particle effects on cut
- Clean, uncluttered UI
- Portrait orientation (mobile-first)

### Audio Design Notes
- Crisp cutting sounds
- Material-specific feedback (soap vs fruit)
- Satisfying "plop" when pieces land
- Coin collection jingle
- Level complete fanfare
