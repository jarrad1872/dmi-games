# DMI Tools Factory

Admin dashboard for managing arcade games, products, and loadouts.

## Development

```bash
pnpm dev
```

## Features (MVP)

- Supabase Auth
- Products CRUD
- Loadouts JSON editor with validation
- Game preview iframe

## Features (v1.0)

- Visual loadout builder
- Drag-drop product selection
- Analytics dashboard
- Promo template gallery
